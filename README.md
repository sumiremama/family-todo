# アプリケーション名
Family-Todo
# アプリケーション概要
* 家族間で共有できるタスク管理アプリ
* 買い物リストや各自の用事を家族間で把握し合う
* 期限付きのタスクは期限も共有することでうっかりを防ぐ
* 投稿者以外のユーザーもリストの編集＆削除ができる
